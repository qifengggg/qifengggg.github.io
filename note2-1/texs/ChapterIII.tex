\section{相关性}

\sssubsection{定义法}

定义法常用于证明相关/无关性。

令 $ \sum k_i\alpha_i = \vec 0, $ 通过
\begin{itemize}
    \item 乘（使等式变短）
    \item 重组
\end{itemize}
得出 $ \forall i, k_i = 0, $ 则 $ \alpha_i $ 线性无关。

\sssubsection{秩}

$$
    \begin{cases}
        r(\alpha_1,\cdots,\alpha_{\Attention{s}}) = \Attention{S} \Rightarrow (\alpha_i)\textrm{线性无关}\\
        r(\alpha_1,\cdots,\alpha_s) < S \Rightarrow (\alpha_i)\textrm{线性相关}\\
    \end{cases}
$$
